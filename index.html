What is https?

Https is short for Hypertext Protocol Secure. The S stands for secure which means that when you visit a https website your information is completely encrypted using Transport Layer Security, TLS, former Secure Sockets Layer, SSL. This is especially important for securing activitites as for example banking or transfers between companies. This helps us to stay safe from people stealing our creditcard numbers etc.

How do I know if a website is using https?
If a website is using https you can see it in the url. A small icon with a padlock marks that the website is using https.
